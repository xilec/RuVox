# SymbolNormalizer

Преобразование операторов и специальных символов в слова.

## Импорт

```python
from ruvox.tts_pipeline.normalizers import SymbolNormalizer
```

## Использование

```python
normalizer = SymbolNormalizer()

normalizer.normalize("->")  # → "стрелка"
normalizer.normalize(">=")  # → "больше или равно"
normalizer.normalize("!=")  # → "не равно"
```

## Операторы

### Стрелки

| Оператор | Результат |
|----------|-----------|
| `->` | стрелка |
| `=>` | толстая стрелка |
| `<-` | стрелка влево |
| `<->` | двунаправленная стрелка |

### Сравнение

| Оператор | Результат |
|----------|-----------|
| `==` | равно равно |
| `===` | строго равно |
| `!=` | не равно |
| `!==` | строго не равно |
| `>=` | больше или равно |
| `<=` | меньше или равно |
| `>` | больше |
| `<` | меньше |

### Логические

| Оператор | Результат |
|----------|-----------|
| `&&` | и |
| `\|\|` | или |
| `!` | не |

### Присваивание

| Оператор | Результат |
|----------|-----------|
| `=` | равно |
| `+=` | плюс равно |
| `-=` | минус равно |
| `*=` | умножить равно |
| `/=` | разделить равно |

### Прочие

| Оператор | Результат |
|----------|-----------|
| `::` | двойное двоеточие |
| `...` | троеточие |
| `??` | два вопроса |
| `?.` | опциональная цепочка |
