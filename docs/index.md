# Fast TTS RUS

**Fast TTS RUS** — приложение для озвучивания технических текстов на русском языке с использованием Silero TTS.

## Проблема

Silero TTS не умеет корректно читать:
- Английские слова и IT-термины (`feature` → молчание или искажение)
- Аббревиатуры (`API`, `HTTP`, `JSON`)
- URL, email, IP-адреса
- Идентификаторы кода (`getUserData`, `my_variable`)
- Спецсимволы и операторы (`->`, `>=`, `!=`)

## Решение

Приложение с desktop UI и встроенным pipeline нормализации:

```
"Вызови getUserData() через API" → "Вызови гет юзер дата через эй пи ай"
```

## Возможности

- **Быстрое чтение** — горячая клавиша для мгновенного озвучивания буфера обмена
- **Очередь чтения** — накопление текстов для последовательного прослушивания
- **Умная нормализация** — автоматическое преобразование технического текста
- **Подсветка слов** — отслеживание текущей позиции в тексте

## Быстрый старт

```bash
# Установка (NixOS)
nix-shell
uv sync --extra ui --extra tts

# Запуск
uv run fast-tts-ui
```

## Технологии

- **Python 3.11+**
- **PyQt6** — desktop UI
- **Silero TTS** — синтез речи
- **num2words** — числа в слова
